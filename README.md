# Network Troubleshooter Simulator

## Overview
The Network Troubleshooter Simulator is a project designed to simulate various network troubleshooting scenarios. This tool is aimed at helping users, network engineers, and students understand common networking issues and how to resolve them effectively.

## Features
- Simulates different types of network problems such as:
  - Connectivity losses
  - IP configuration issues
  - DNS errors
  - Network congestion
- Provides guided troubleshooting steps to resolve each issue.
- User-friendly interface to allow easy simulation of scenarios.
