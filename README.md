# Network Troubleshooter Simulator

## Play the Game
[Click here to play the game](http://localhost:3000)

## Installation Steps
1. Clone the repository: 
   ```bash
   git clone https://github.com/Pratik12mukh/network-troubleshooter-simulator.git
   ```
2. Change directory into the project folder:
   ```bash
   cd network-troubleshooter-simulator
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the application:
   ```bash
   npm start
   ```

## Game Levels
- **Level 1**: Basic connectivity issues.
- **Level 2**: Advanced network configuration.
- **Level 3**: Troubleshooting real-world scenarios.

## Features
- Interactive game interface.
- Step-by-step guidance through troubleshooting.
- Score tracking and performance analytics.
